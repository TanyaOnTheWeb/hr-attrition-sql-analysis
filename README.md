# HR Attrition Analysis — SQL + Power BI

## Problem Statement
Why are employees leaving? This project analyses 1,470 employee 
records to identify the strongest drivers of attrition using SQL.

## Tools Used
MySQL · Power BI · GitHub

## Dataset
IBM HR Analytics — Kaggle (1,470 rows, 35 columns)

## Key Findings
- Overall attrition rate: 16.12% (above 10–12% benchmark)
- Overtime employees leave at 3X the rate of non-overtime (30.5% vs 10.4%)
- Sales Representatives: 40% attrition — highest risk role
- Year 1 new hires: 28% attrition — first year is the danger zone
- High-risk cohort (4 combined signals): 50–75% attrition

## SQL Concepts Demonstrated
- SELECT, FROM, WHERE, GROUP BY, ORDER BY
- COUNT, SUM, AVG, ROUND, CASE WHEN
- CTEs (WITH clause)
- Window Functions — RANK(), ROW_NUMBER(), running totals
- Correlated subqueries
- Pivot tables using CASE WHEN
- Weighted risk scoring model in pure SQL
